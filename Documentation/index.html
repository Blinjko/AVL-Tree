AVLTree Class

The AVLTree.h file contains the definition of the AVLTree class, and the implementation of the class also. The AVLTree class is inside the DataStructures namespace, so to make a new AVL-Tree you would need to the following DataStructures::AVLTree<type>. The AVL-Tree class is a templated class, which allows for more type flexiblily, however whatever type is used must be comparable type, and have the <, >, and == operator overloaded.
